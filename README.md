# ved AI - Virtual Desktop Assistant 🚀

A Python-based personal assistant powered by **Groq Cloud API (Llama 3.1)** for intelligent conversations and system automation.

* **B.Sc. (H) Mathematics** - Sri Venkateswara College, Delhi University.

## ✨ Features
- **Smart Conversations:** Powered by Llama-3.1-8b model via Groq API.
- **System Automation:** Control Volume (Up/Down), Shutdown, and Tab management (Chrome/Edge/Brave).
- **Web Controls:** Quick voice access to YouTube, Google, and WhatsApp.
- **Real-time Updates:** Fetches live weather updates for Delhi and current IST time.
- **Standby Mode:** Controlled by "ved On/Off" wake-word logic to prevent accidental triggers.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **AI Model:** Llama-3.1-8b-instant (via Groq API)
- **Libraries:** - `SpeechRecognition` (Voice Input)
  - `Pyttsx3` (Text-to-Speech)
  - `PyAutoGUI` (System Automation)
  - `Requests` (API calls)
